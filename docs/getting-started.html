Getting Started

Pre-reqs

  • Wallet (MetaMask / Coinbase / WalletConnect)
  • RPC access to your target chain (Hedera testnet by default)
  • Web3.Storage token for IPFS

Deploy Contracts

cd contracts
export RPC_URL="https://testnet.hashio.io/api"
export PRIVATE_KEY=0x...                    # deployer
export ROYALTY_RECEIVER=0xYourTreasury
export DEFAULT_ROYALTY_BPS=500
forge build
forge script script/Deploy.s.sol:Deploy --rpc-url $RPC_URL --broadcast -vvvv

Run the dApp

cd dapp
cp .env.example .env
# Fill VITE_CHAIN_ID, VITE_RPC_URL, VITE_CONTRACT_ADDRESS, VITE_REGISTRY_ADDRESS, VITE_WEB3STORAGE_TOKEN, VITE_WALLETCONNECT_PROJECT_ID
npm i
npm run dev

Mint your first item

  1. Connect your wallet.
  2. Use AI or upload content via Create / Mint.
  3. Click Mint NFT.
  4. Find it in Gallery and explore forks in Fork Explorer.
