FAQ

What wallets are supported?

Injected wallets (MetaMask), Coinbase Wallet, and WalletConnect.

Where is content stored?

On IPFS via Web3.Storage, with optional Pinata pinning. Metadata includes SHA-256 hashes and provenance.

Is there moderation?

Yes. Client-level NSFW filter and a minimal IPFS report file system. Admins can hide/unhide via tokenURI updates.
